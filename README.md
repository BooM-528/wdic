# Why did I CALL (WDIC)

Welcome to **Why did I CALL (WDIC)**!

WDIC is a web application designed to store and manage poker tournament history sessions and provide advanced gameplay analysis using AI. By leveraging artificial intelligence, players can review and analyze their decisions ("Why did I call?") to continuously improve their poker strategies.

## 🚀 Project Overview

The project is structured as a monorepo containing both the frontend and the backend:

- **`/frontend`**: The user interface built with modern web technologies (Next.js/React).
- **`/backend`**: The core API, database management, and AI analysis engine (Django/Python).
